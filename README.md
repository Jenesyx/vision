### Run following commands to use Front-end

==> In website-app folder

```
yarn add react
```

==> In Backend folder

```
npm i express
npm i cors
```

==> Start the project with
```
yarn start
```

==> Start Api in Backend folder
```
node server.js
```


Make sure to start the xampp and connect it with the write name
